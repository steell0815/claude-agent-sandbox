# Capabilities Inventory

List everything available in this session. Present as a compact inventory:

## What to Report

1. **Built-in Tools**: List every tool you have access to (Read, Write, Edit, Bash, Grep, Glob, Agent, etc.)
2. **Deferred Tools**: List any deferred tools available via ToolSearch
3. **Skills / Slash Commands**: List all `/command` skills you can see, with their descriptions
4. **MCP Servers & Tools**: List any MCP servers configured and their tools
5. **Hooks**: List configured hook events and what they do
6. **Agents**: List available agent types (Explore, Plan, etc.)
7. **Model**: Current model name

Present as a compact, scannable list grouped by category. Use tables where appropriate. No lengthy explanations needed.
